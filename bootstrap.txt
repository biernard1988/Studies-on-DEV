A grade do Bootstrap!

Vamos revisar alguns conceitos-chave:

Em caso de dúvida, consulte a documentação do Bootstrap.

Há alguns links necessários para usar o Bootstrap (o arquivo CSS e duas tags <meta>)

O Bootstrap 4 tem um sistema de grade implementado usando flexbox

O sistema de grade é composto de contêineres, linhas e colunas que trabalham juntos para criar o layout de uma página da Web.

Os contêineres são necessários para implementar a grade.

Os contêineres contêm linhas que contêm colunas.

A grade do Bootstrap segue um sistema de 12 colunas.

O Bootstrap usa design responsivo e foi criado com base em pontos de interrupção relacionados aos tamanhos de tela dos dispositivos.

Para definir manualmente a largura de uma coluna, precisamos seguir a convenção de nomenclatura do Bootstrap.

Podemos adicionar várias classes a uma coluna para determinar a largura que uma coluna deve ter em pontos de interrupção específicos.
